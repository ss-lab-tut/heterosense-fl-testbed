# Figure Generation Scripts

All figures in the HeteroSense-FL paper are generated by these scripts.
They use only HeteroSense-FL itself — no external data, no pre-baked numbers
(except Fig. 1 and Fig. 4 which are schematic diagrams with no data dependency).

## Prerequisites

```bash
pip install heterosense-fl matplotlib
```

## Reproduce all figures

```bash
cd scripts/figures

python generate_fig1_architecture.py     # Architecture diagram  → /tmp/fig1_architecture.png
python generate_fig2_modality_bundles.py # Observation examples  → /tmp/fig2_modality_bundles.png
python generate_fig3_benchmark.py        # Table 3 bar chart     → /tmp/fig3_benchmark.png  (~3 min)
python generate_fig4_temporal_window.py  # Sampler schematic     → /tmp/fig4_temporal_window.png
python generate_fig5_comparison.py       # Feature-comparison    → /tmp/fig5_comparison.png
```

## Correspondence to paper

| Script | Paper figure | Data source |
|--------|-------------|-------------|
| `generate_fig1_architecture.py` | Fig. 1 | Schematic (no data) |
| `generate_fig2_modality_bundles.py` | Fig. 2 | Generated by `DatasetBuilder` |
| `generate_fig3_benchmark.py` | Fig. 3 | Computed by `heterosense-benchmark` (seeds {42,123,7}) |
| `generate_fig4_temporal_window.py` | Fig. 4 | Schematic (no data) |
| `generate_fig5_comparison.py` | (supplementary) | Generated by `DatasetBuilder` |

## Numerical consistency

Fig. 3 computes the same benchmark as `heterosense-benchmark` using
identical seeds, n_steps, rounds, and encoder. The numbers in Fig. 3
match Table 3 in the paper exactly.

To verify:

```bash
python generate_fig3_benchmark.py   # prints computed numbers
heterosense-benchmark               # prints same numbers
```
